# AirMouse

A computer vision project that uses a webcam to track hand movements
and control the mouse cursor.

## Features
- Hand tracking using camera
- Cursor movement using hand gestures
- Click / drag support (if any)

## Tech Stack
- Python
- OpenCV
- MediaPipe

## How to Run
1. Install Python
2. Install dependencies:
   pip install opencv-python mediapipe pyautogui
3. Run:
   python airmouse.py
